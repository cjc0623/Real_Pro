## 📖 프로젝트 소개

FIRST ROAD는 화주와 운송사를 연결해주는  
**화물 운송 중개 플랫폼**입니다.

- 화주는 화물을 등록하고
- 운송사는 해당 화물에 입찰을 진행하며
- 최종적으로 매칭을 통해 배송이 이루어집니다.

Spring Boot 기반 REST API 서버로 구현되었으며,  
React 프론트엔드와 연동됩니다.

## ✨ 주요 기능

### 🔐 1. 회원 관리
- 회원가입 / 로그인 기능 제공
- JWT 기반 인증 처리
- 사용자 권한별 접근 제어

---

### 🚛 2. 화물 등록
- 화주가 화물 정보 등록 가능
- 출발지 / 도착지 / 무게 / 상세 정보 입력

---

### 💰 3. 입찰 시스템
- 운송사가 화물에 입찰 가능
- 가격 기반 경쟁 구조

---

### 📦 4. 배송 관리
- 배송 상태 조회 (대기 / 진행 / 완료)
- 배송 흐름 관리

---

### 📬 5. 문의 게시판
- 사용자 문의 등록
- 문의 내역 조회
- 관리자 응답 확장 가능
